# Fakturomat
The project was created as part of the programming course.

Fakturomat is a tool for invoice management.

#### Applied technologies:
- Maven
- JavaFx
- OrmLite
- H2 Database
- JUnit 4

## Features!
- Add Products
- Define clientsa and sellers
- Create and view invoices
